# PetShop Amigo Fiel

Este é um projeto acadêmico desenvolvido para a disciplina **Fundamentos de Sistemas Web**.

## Fase 1
O objetivo foi criar um site simples e funcional para um petshop utilizando apenas HTML, sem o uso de CSS, JavaScript ou frameworks como Bootstrap.

## Fase 2
Nesta fase, o site foi aprimorado com:
- **CSS/Bootstrap**: Carrossel de imagens, cards responsivos e layout moderno
- **JavaScript**: Função temporal (mensagem de bom dia/tarde/noite), validação de formulários
- **Formulário de Cadastro**: Dados do cliente (nome, CPF, endereço, sexo, telefone, e-mail) e dados do pet (nome, raça, idade)
- **Agendamento**: Escolha de serviço (banho/tosa), método (tele-busca/entrega no local), calendário para data e horário
- **Acessibilidade**: Atributo `alt` em todas as imagens para audiodescrição

## Estrutura do Site

| Página | Descrição |
|--------|-----------|
| `index.html` | Página inicial com carrossel e mensagem temporal |
| `produtos.html` | Produtos divididos em categorias (rações, brinquedos, acessórios) |
| `servicos.html` | Serviços oferecidos (banho, tosa, consulta) |
| `cadastro.html` | Formulário de cadastro de cliente e pet |
| `agendamento.html` | Formulário para agendamento de serviços |
| `contato.html` | Informações de contato |

## Tecnologias Utilizadas
- HTML5
- CSS3 / Bootstrap 5
- JavaScript

## Acessibilidade
Todas as imagens possuem atributo `alt` com descrições detalhadas para leitores de tela.

Sobre o site

O site é composto por quatro páginas, todas interligadas por um menu de navegação comum:

    Home (index.html): Apresenta o petshop e dá uma visão geral das categorias de produtos disponíveis.

    Produtos (produtos.html): Exibe dois produtos de cada uma das três categorias (Rações, Brinquedos e Acessórios), com imagem, descrição e preço.

    Serviços (servicos.html): Lista os serviços oferecidos, como Banho, Tosa e Consulta Veterinária. Cada serviço inclui descrição, valor e indicação se há opção de tele-busca.

    Contato (contato.html): Traz informações básicas de contato do estabelecimento, como telefone, e-mail e endereço.
